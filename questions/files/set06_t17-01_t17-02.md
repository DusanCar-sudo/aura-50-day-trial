# Question Set 06 — t17-01, t17-02

## t17-01
A session has turn 1: 6,595 input tokens, 1,280 cached. Turn 4: 7,717 input, 7,680 cached. Calculate the actual new tokens billed in each turn, explain why turn 4 costs almost nothing despite having more total tokens, and estimate what turn 10 would cost.

## t17-02
A tool call returns a 50,000-character ThreadSanitizer race condition report. This gets appended to conversation history. Explain the exact mechanism by which this single tool result causes every subsequent turn to cost significantly more, even if the user only types short messages.
